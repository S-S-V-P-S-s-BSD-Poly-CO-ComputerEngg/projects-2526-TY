# Gas Agency Management System

## 1. Project Title

Gas Agency Management System

---

## 2. Project Description

Gas Agency Management System is a web-based MERN stack application developed to manage customer bookings, cylinder delivery, stock, and payments.
This system helps automate daily gas agency operations with an easy-to-use admin and customer interface.
