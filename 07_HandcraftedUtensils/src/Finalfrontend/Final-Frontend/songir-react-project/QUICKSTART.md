# 🚀 Quick Start Guide - Songir Marketplace

## ⚡ 3-Step Setup

### 1️⃣ Install Dependencies
```bash
cd songir-react-project
npm install
```

### 2️⃣ Run the App
```bash
npm start
```

### 3️⃣ Open Browser
Visit: **http://localhost:3000**

## 🐛 Troubleshooting

**Port 3000 in use?**
```bash
PORT=3001 npm start
```

**Errors after install?**
```bash
rm -rf node_modules
npm install
```

**Need help?**
- Check INSTALLATION.md
- Review CONVERSION_NOTES.md
- Inspect the code - it's well commented!

## 💡 Pro Tips

✨ **To add a new product:**
Edit `src/data/mockData.js` → Add to `products` array

🎨 **To change colors:**
Edit `src/styles/App.css` → Modify `:root` variables

📄 **To add a new page:**
1. Create file in `src/pages/`
2. Add route in `src/App.js`
3. Add link in Navbar
